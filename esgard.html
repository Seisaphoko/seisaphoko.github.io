Esgard Store - Retail Management System

Project Overview

Esgard Store is a comprehensive point-of-sale and inventory management system built for Esgard, an exclusive clothing store. The platform combines retail operations management with customer relationship features in one integrated solution.

Key Features

  • Point-of-Sale: Process customer transactions efficiently
  • User Authentication: Secure registration and login system
  • Access Control: Role-based permissions for customers, staff, and admins
  • Loyalty Points Program: Rewards system for repeat customers
  • Inventory Management: Track stock levels and product variations
  • Payment Processing: Secure payment integration

System Capabilities

  • Real-time inventory tracking
  • Customer account management
  • Sales reporting and analytics
  • Product catalog management
  • Staff management and permissions
  • Transaction history and audit trails

Technologies Used

  • Frontend: Visual Studio 2019]
  • Backend: C++ on visual studio
  • Database: SQL
